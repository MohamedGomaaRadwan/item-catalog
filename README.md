- lotsofmenus.py    -> sample data used (from instructor notes)

# Requirments

- Virtual Box
- Vagrant
- Python
- Flask

# Let's run our app

- cd vagrant and type "vagrant up;vagrantssh" 
- python lotsofmenus.py
- python project.py
- The app will be on localhost:8000
